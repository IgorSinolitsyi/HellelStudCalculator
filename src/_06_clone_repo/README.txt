1) Клонировать проект с GitHub можем
непосредственно в папку проектов.

Или, в папке проектов, создать отдельную папку 
для этого, например, AAB .


2) Заходим на GitHub, находим нужный проект.
Или, входим в GitHub-Calculator-IDE, который
склонируем в отдельную папку.

Жмем кнопку Code, и, в выпадающем окне, кликаем 
на иконку копирования (URL проекта скопирован в буфер, 
можем сохранить у себя на машине).


3) Запускаем IDE, и жмем кнопку Get from VCS .

В появившемся окне, напротив Directory, выбираем
место/папку куда будем клонировать репозиторий.

Например,
C:\Users\Admin\AAB

Напротив URL вставляем URL репозитория с GitHub.

Например, 
https://github.com/admin/Git-app.git

Также, допишем путь размещения клонируемого 
репозитория

Например,
C:\Users\Admin\AAB\Git-app

Жмем кнопку  Clone .

Репозиторий клонируется на Вашу машину.
IDE запускает проект.


4) Исследуйте проект. Просмотрите коммиты.


5) ЕСЛИ находитесь в уже открытом в IDE проекте,
в верхнем меню

File>New>Project from Version Control...

Появится такое же окно для подготовки клонирования репозитория.
Действия - те же.
